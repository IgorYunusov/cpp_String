Detailed Description

A custom String implementation.

A custom C++ implementation of a String object for pedagogical purposes. Is efficient for specifying the length of your String and does not depend on the use of a NULL termination.

Author
Bryan Goodrich
Version
0.2
Date
2013-12-13
Invariant
String data will always point to NULL or a valid address.
String length will always indicate amount of allocated space.
